# Threello 프로젝트

- 트렐로와 같은 칸반보드 서비스 기능 구현
- 내용 : 칸반보드 워크플로우 서비스 기능을 Nest.js와 TypeORM으로 구현합니다.

## 👋 Threello 팀 소개

- **Threello**는 3조인 저희 조 번호와 트렐로를 합친 이름입니다.

- **손민영**
